# Statistical Models - University Project

## 📌 Overview
This repository contains a university project developed for the **Statistical Models** exam. 

## 📊 Dataset
The analysis utilizes the publicly available **SUPPORT2** (Study to Understand Prognoses and Preferences for Outcomes and Risks of Treatments) dataset.

## 📂 Contents
* `support2_disability_models.ipynb`: The complete Jupyter Notebook containing the source code, feature embedding pipelines, and model evaluation.
* `abstract_SUPPORT2_project.pdf`: The comprehensive project abstract detailing the clinical background, methodology, and results.
